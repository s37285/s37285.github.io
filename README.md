# s37285.github.io
ยินดีต้อนรับเข้าสู่เว็บไซต์แฟ้มสะสมงานของนางสาวธันยพร หลวงจิตร
สวัสดีค่ะดิฉันนางสาวธันยพร หลวงจิตร ชื่อเล่น เมล
อายุ 17 ปี กำลังศึกษาอยู่ชั้น ม.6 แผนการเรียน gifted โรงเรียนบ้านบึงอุตสาหกรรมนุเคราะห์
ดิฉันชอบในรายวิชา ชีววิทยา
งานอดิเรกของฉัน คือ ชอบดูหนัง ฟังเพลง และอ่านหนังสือ
ดิฉันเป็นคนสดใส ร่าเริง ยิ้มง่ายทำให้เข้าับผู้อื่นได้อย่างรวดเร็ว
ดิฉันมีทักษะในการทำงานเป็นทีมได้เป็นอย่างดี
ในอนคตดิฉันอยากเข้าศึกษาในคณะเภสัชศาสตร์ มหาลัยมหิดล
แรงบันดาลใจในการเรียนของดิฉัน คือ เชื่อว่าความพยายามและการเรียนรู้อย่างต่อเนื่องจะนำไปสู่ความสำเร็จ
เป้าหมายในตอนนี้ของดิฉัน คือ อยากพัฒนาความรู้ในวิชาที่ถนัดและปรับปรุงวิชาที่ยังไม่เข้าใจ
ดิฉันมีคติประจำใจ คือ ทุกความสำเร็จเริ่มต้นจากการลงมือทำ
